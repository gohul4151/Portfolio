# Gohul S — Portfolio Website

A personal portfolio website built with vanilla HTML, CSS, and JavaScript. Features a dark-themed design with animated particles, smooth scroll interactions, and responsive layout.

**Live:** [portfolio-blond-five-94.vercel.app](https://portfolio-blond-five-94.vercel.app/)

---

## Features

- **Animated particle canvas** — floating particles with connecting lines in the background
- **Typing effect** — cycling role titles in the hero section
- **Cursor glow** — smooth cursor-following light effect
- **Scroll reveal** — elements animate in as you scroll
- **3D tilt cards** — project cards respond to mouse movement
- **Counter animation** — stats count up when scrolled into view
- **Responsive design** — mobile-friendly with hamburger nav
- **Active nav tracking** — highlights current section in navbar
